# PayFlow SaaS

Sistema SaaS para gestión de proveedores, facturas y flujo de aprobación.

## Tecnologías

- Node.js
- Express
- PostgreSQL
- JWT
- Docker

## Funcionalidades actuales

- Registro de empresa
- Login
- JWT Authentication
- Multiempresa
- Endpoint protegido

## Instalación

```bash
npm install
npm run dev
```

## Variables de entorno

```env
PORT=4000
DB_HOST=127.0.0.1
DB_PORT=5433
DB_NAME=payflow_saas
DB_USER=postgres
DB_PASSWORD=*****
JWT_SECRET=*****
```
